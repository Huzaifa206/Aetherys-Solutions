## This Website

This site **is** our portfolio. Built from scratch using:

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** CSS Modules with a custom design system
- **Animations:** Vanilla CSS keyframes + Intersection Observer API
- **Email:** EmailJS (contact form → real inbox delivery)
- **Design Language:** Brutalist typography · Cyberpunk minimalism · Motion-first UI

Every pixel, animation, and interaction was intentionally crafted to demonstrate what we can build for *you*.

## Getting Started (Local Dev)

```bash
# Clone the repo
git clone https://github.com/aetherys/agency-website.git
cd agency-website

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
